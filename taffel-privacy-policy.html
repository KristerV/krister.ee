Privacy Policy - Tahvel Auto-Fill Extension

Last Updated: November 20, 2025

Simple Summary: This extension does not collect, store, or transmit any personal data. It operates entirely within your browser to add automation features to the Tahvel education system.

1. What This Extension Does

Tahvel Auto-Fill is a browser extension that adds automation features to the Tahvel education system (tahvel.edu.ee). It provides:

  • Quick navigation buttons for common Tahvel pages
  • Bulk actions for marking student attendance
  • Automated lesson date filling
  • Tinkr integration for attendance tracking

2. Data Collection

We do not collect any data.

This extension:

  • Does not collect personal information
  • Does not track your browsing activity
  • Does not store data on external servers
  • Does not use analytics or tracking services
  • Does not transmit any information to third parties

3. How the Extension Works

The extension operates entirely within your browser:

  • Local Operation: All code runs on your device
  • Tahvel API Access: Communicates only with tahvel.edu.ee to perform actions you initiate
  • No External Connections: Does not connect to any servers other than tahvel.edu.ee
  • Session-Based: Uses your existing Tahvel login session; does not handle or store credentials

4. Permissions Explained

The extension requests the following permissions:

activeTab

Allows the extension to interact with the active Tahvel page to inject automation buttons and features into the interface.

Host Permission: tahvel.edu.ee

The extension only works on tahvel.edu.ee. This permission allows it to:

  • Add automation features to journal pages
  • Create navigation buttons in the interface
  • Read and update lesson attendance data through the Tahvel API

The extension cannot access any other websites or data outside of tahvel.edu.ee.

5. Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

6. Data Security

Since no data is collected or transmitted, there are no data security concerns beyond your standard Tahvel account security. Always:

  • Keep your Tahvel credentials secure
  • Log out of Tahvel when using shared computers
  • Download extensions only from official browser stores

7. Changes to This Policy
