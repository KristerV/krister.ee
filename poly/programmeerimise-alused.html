Programmeerimise Alused

Gruppidele MM24, TA24A ja TA24B.

Taustinfo

Tarkvara ja internet on lahutamatud - ilma internetita oleks tänapäevane tarkvara sama kasulik kui trükimasin. Internet ja andmebaasid on kesksel kohal tänapäeva tehnoloogias, mistõttu on need ka meie kursuse olulised komponendid.

Aine Ülevaade

Kursus koosneb kolmest projektist:

  1. Projekt 0 (Tutorial): Chati rakenduse loomine koos õpetajaga
  2. Projekt 1: Veebilehe loomine freepublicapis.com API põhjal
  3. Projekt 2: Vabalt valitud teema etteantud nõuetega

Olulised Lingid

  • Chat rakendus: CodePen, GitHub
  • Toolkit: CodePen, GitHub

Projekt 1

Projekt 1 ülesanne on valida omale freepublicapis.com lehelt API ja selle API datast teha veebileht.

Nõuded

  • JavaScript laeb andmed alla kasutades fetch() funktsiooni
  • Andmed pannakse lehele dünaamiliselt
  • Andmeid on vähemalt 10 rida

Projekti video õpetus on siin: Projekt 1 Youtube tutorial

Projekt 2

Nõuded

  • Kasutaja saab sisestada andmeid
  • Andmed salvestuvad andmebaasi
  • Andmed kuvatakse lehel
  • Rakendust saavad kasutada mitu kasutajat korraga
  • Rakendus on avalikult kättesaadav internetis (GitHub Pages)
  • Rakendus on kellelegi kasulik

Hinde saamiseks küsin ma koodi kohta küsimusi ja iga grupi liige peab suutma selle tööd selgitada.

Pane oma projekti idee kirja Excelisse: Projektide Registreerimise Tabel

Näidis ideed

Järgnevalt mõned ideed projekti 2 jaoks:

  • Õpivahend kontrolltöödeks: Näiteks keeleõppe rakendus, kus kasutaja saab sisestada sõnapaare (eesti ja võõrkeeles). Programm küsib neid sõnu ja hindab vastuseid.
  • Mäng: Võid luua mõne klassikalise mängu nagu Tetris (alustades lihtsustatud versioonist 1x1 plokkidega) või Trips-Traps-Trull.
  • Veebilehe kloon: Mõne olemasoleva veebilehe või rakenduse oma versioon, kus saad rakendada oma nägemust paremast lahendusest.

Hindamisel arvestatakse JavaScripti keerukust. Edasijõudnud õpilased võivad kasutada ka teisi programmeerimiskeeli.

Kui sul puuduvad ideed, võta julgelt ühendust ja arutame koos läbi sobiva projekti.

Andmebaasi Kasutamine

Andmebaasi URL:

https://kool.krister.ee/chat/SINU_PROJEKTI_NIMI

NB! Asenda SINU_PROJEKTI_NIMI oma projekti nimega.

Võta meie tehtud Chati rakendus lahti ja vaata kuidas sinna andmeid saata ja kuidas sealt andmeid küsida. SINU_PROJEKTI_NIMI võimaldab sul täiesti oma andmebaasi tekitada. Pane sinna mis andmeid tahad, kasuta isegi mitut nime, kui projekt vajab mitut andmebaasi tabelit.

See andmebaas EI OLE turvaline. Ära salvesta sinna paroole ega isikuandmeid. Vahel andmed kustuvad lihtsalt ära.
